# vscode 支持

## 介绍

Concis 针对 vscode 开发工具提供了代码自动补全、语法高亮、提示信息的功能，可以在插件市场中下载：

<img src="https://concis.org.cn/images/vscode.plugin.jpg" height="600" />

## 使用

下载完毕后，通过 c + 组件名称，如 cbutton 即可快速生成 Button 组件。
